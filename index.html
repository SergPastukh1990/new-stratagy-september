Что такое «коробка»?

Коробка — это заранее упакованный фиксированный сервис по AI, который:

  • ✓ Имеет чёткий скоуп (что входит, что не входит)
  • ✓ Фиксированное время исполнения (1,5–2 недели)
  • ✓ Фиксированная стоимость ($5–10k)
  • ✓ Решает конкретную бизнес-задачу
  • ✓ Точка входа для дальнейших апсейлов

Примеры коробок:

AI Process Audit & Optimization

  • Проведение сессии по оценке потенциала оптимизации бизнес-процессов через AI
  • Выход: список quick-win автоматизаций, roadmap внедрения
  • Срок: ~20 часов (2 недели)
~$5k

Automation Sprint (n8n / Make / Zapier)

  • Настройка автоматизации рабочих процессов (CRM, отчёты, e-mail/Slack уведомления)
  • Выход: работающий сценарий автоматизации, обученная команда клиента
  • Срок: 10–15 дней
$4–6k

AI Copilot для отдела продаж / логистики

  • Внедрение чат-бота/агента для ускорения внутренних процессов
  • Выход: работающий бот + инструкция по использованию
  • Срок: 2 недели
$6–7k

Docs2Data (AI для документов)

  • Автоматическая обработка накладных, инвойсов, актов
  • Выход: RAG-пайплайн, выгружающий данные в систему клиента
  • Срок: 2 недели
$7–9k
